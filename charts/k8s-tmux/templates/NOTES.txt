1. Get the application URL by running these commands:
{{- if .Values.ingress.enabled }}
{{- range $host := .Values.ingress.hosts }}
  {{- range .paths }}
  http{{ if $.Values.ingress.tls }}s{{ end }}://{{ $host.host }}{{ .path }}
  {{- end }}
{{- end }}
{{- else if contains "NodePort" .Values.service.type }}
  export NODE_PORT=$(kubectl get --namespace {{ .Release.Namespace }} -o jsonpath="{.spec.ports[0].nodePort}" services {{ include "k8s-tmux.fullname" . }})
  export NODE_IP=$(kubectl get nodes --namespace {{ .Release.Namespace }} -o jsonpath="{.items[0].status.addresses[0].address}")
  echo http://$NODE_IP:$NODE_PORT
{{- else if contains "LoadBalancer" .Values.service.type }}
     NOTE: It may take a few minutes for the LoadBalancer IP to be available.
           You can watch the status of by running 'kubectl get --namespace {{ .Release.Namespace }} svc -w {{ include "k8s-tmux.fullname" . }}'
  export SERVICE_IP=$(kubectl get svc --namespace {{ .Release.Namespace }} {{ include "k8s-tmux.fullname" . }} --template "{{"{{ range (index .status.loadBalancer.ingress 0) }}{{.}}{{ end }}"}}")
  echo http://$SERVICE_IP:{{ .Values.service.port }}
{{- else if contains "ClusterIP" .Values.service.type }}
  export POD_NAME=$(kubectl get pods --namespace {{ .Release.Namespace }} -l "app.kubernetes.io/name={{ include "k8s-tmux.name" . }},app.kubernetes.io/instance={{ .Release.Name }}" -o jsonpath="{.items[0].metadata.name}")
  export CONTAINER_PORT=$(kubectl get pod --namespace {{ .Release.Namespace }} $POD_NAME -o jsonpath="{.spec.containers[0].ports[0].containerPort}")
  echo "Visit http://127.0.0.1:7681 to use your application"
  kubectl --namespace {{ .Release.Namespace }} port-forward $POD_NAME 7681:$CONTAINER_PORT
{{- end }}

2. Access your k8s-tmux development environment:
   - The web terminal will be available at the URL above
   - Default tmux session 'ai-dev' with three windows:
     * Window 0: Main development workspace
     * Window 1: Claude AI interaction workspace  
     * Window 2: File management workspace

3. Kubeconfig Setup:
   Place your kubeconfig file at one of these locations in the shared storage:
   - /mnt/k8s-tmux/.kube/config (recommended)
   - /mnt/WiredQuill/.kube/config
   - /mnt/k8s-tmux/config
   - /mnt/WiredQuill/config

4. Available Tools:
   - Kubernetes: kubectl (k), kubectx (kx), kubens (kn), k9s, helm
   - System: btop, mc, tree, fastfetch
   - Editors: vim, nano
   - Shell: bash (with completion), zsh (oh-my-zsh)

{{- end }}
{{- if .Values.persistence.wiredQuill.enabled }}
   - WiredQuill storage: {{ .Values.persistence.wiredQuill.nfs.server }}:{{ .Values.persistence.wiredQuill.nfs.path }}

6. To view logs:
   kubectl logs --namespace {{ .Release.Namespace }} -l "app.kubernetes.io/name={{ include "k8s-tmux.name" . }},app.kubernetes.io/instance={{ .Release.Name }}"
